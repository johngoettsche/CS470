Release Notes

Version 1.2.1 - released 18.12.2013

  • implemented EProfiler static library alternative for all platforms
  • updated user guide documentation

Version 1.2.0 - released 18.11.2013

  • implemented full support for 64-bits Windows, MSVC compiler, Intel processor
  • implemented full support for 64-bits Windows, MinGW compiler, Intel processor
  • implemented command line arguments for Performance Analyzer
  • implemented persistent settings for profiler overhead in Performance Analyzer
  • implemented accepting map files directly instead of symbol file in Performance Analyzer
  • improved open dialog in Performance Analyzer
  • fixed bug which prevents to show function names when multiple modules are profiled
  • updated user guide documentation

Version 1.1.1 - released 26.11.2012

  • launched forum application at our website
  • added new samples to demonstrate profiling in easy way
  • fixed very slow loading of big symbol files
  • minor changes in documentation

Version 1.1.0 - released 7.11.2012

  • implemented new Function List Mode into EProfiler
  • implemented support for new EProfiler log format into Performance Analyzer
  • removed log format selection in Performance Analyzer
  • moved display cycle count flag in Performance Analyzer into persistence storage
  • fixed Performance Analyzer crash in case of no memory
  • updated user guide documentation

Version 1.0.2 - released 5.9.2012

  • fixed EProfilerSymGen for MSVC not to mix code section with static symbols (bad call tree was shown)
  • fixed showing of recursive functions in Performance Analyzer for MSVC
  • minor changes in documentation

Version 1.0.1 - released 15.8.2012

  • fixed overflowed wall times in Performance Analyzer
  • fixed bad exit address resolution for MSVC
  • improved documentation

Version 1.0.0 - released 10.8.2012

  • implemented full support for 64-bits Linux, gcc compiler, Intel processor
  • implemented EProfiler interface for C language
  • implemented symbol generator command line tool
  • implemented profiler log conversion command line tool
  • improved Performance Analyzer GUI
  • improved default sorting of columns in Performance Analyzer views
  • added "Show in view" feature to Performance Analyzer
  • added estimates of function durations as if they were not profiled
  • reduced memory requirements of Performance Analyzer
  • fixed auto profiling mode for Android
  • fixed auto profiling mode for Windows XP using MinGW compiler
  • fixed MSVC return addresses
  • fixed several unresolved symbols for GCC logging
  • fixed CPU frequency detection

Version 0.1.0 - released 18.6.2012

  • implemented the first EProfiler version with Performance Analyzer viewer
  • implemented support for Android 2.3.4, gcc compiler, ARM processor (without Performance Analyzer)
  • implemented support for 32-bits Linux, gcc compiler, Intel processor
  • implemented support for 64-bits Linux, gcc compiler, Intel processor (Performance Analyzer only)
  • implemented support for 32-bits Windows, MinGW compiler, Intel processor
  • implemented support for 32-bits Windows, MSVC 9.0 compiler, Intel processor
  • implemented support for Windows CE 5.0, MSVC 9.0 compiler, SH4 processor (without Performance Analyzer)
